Word Wide Web General by Chris Bachl

What are the most essential technologies that made the World Wide Web such a success?

The most essential technologies are URL, URI, HTML, HTTP.

How are resources (html pages, images, etc) addressed in the web?

With a protocol (http), a hostname (www.example.com), and a file name (index.html).

What are the general components of a url?

A Uniform Resource Locator (URL or web address) has a scheme that includes an information about what it is: a file or a news. Considering what it is there will be a specific part, like the username, host, port, path, query, fragment in case the URL starts with Https://

Web Server

What is a web server?

A Web server is a computer with webserver software

What does a web server?

The primary function of a web server is to store, process and deliver web pages to clients.

How can content be stored on a web server?

The files that are stored on a webserver are stored in a data base or simply as a file on the desktop.

How is content in webpages described

Web pages delivered are most frequently HTML documents, which may include images, style sheets and scripts in addition to text content.

Web Client

What is a web client

A web client is every device that is connected to the internet.

What does a web client?

A web client sends data to servers that will analyze the request and serve the client with the requested information.

Web Client and Web Browser is the same?

No, as I Sayed the client will only send and receives packages, and the web browser will display this content and manage what the client should request next from the server.

What is the html head element?

the element is a container for metadata (data about data) and is placed between the tag and the tag.

What does <!DOCTYPE`> html mean?

The declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in.

What are the two top most elements within the html element?

The Body element and the head element

How to emphasize a text and how is it displayed in the browser?

The <em> tag is a phrase tag. It renders as emphasized text.

How to mark up important ext and how is it displayed in the browser?

The <strong> tag is a phrase tag. It defines important text.
